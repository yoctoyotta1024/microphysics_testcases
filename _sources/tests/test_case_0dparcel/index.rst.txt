0-D Parcel Test Case
====================

This test case models a (0-D) parcel/volume of air as it adiabatically
expands and contracts given an arbitrary microphysics scheme.
See :doc:`adiabatic_motion` for the physical equations modelled.

See `tests/test_case_0dparcel/test_mock_py.py <https://github.com/yoctoyotta1024/microphysics_testcases/blob/main/tests/test_case_0dparcel/test_mock_py.py>`_`
for an example of this test case working.
