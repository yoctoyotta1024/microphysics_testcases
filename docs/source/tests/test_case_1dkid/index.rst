1-D KiD Test Case
====================

This test case models a (1-D) rainshaft based on the KiD-A
intecomparison test case. See more details on the KiD model from
:cite:`Hilletal2023`, :cite:`ShipwayHill2012`, and :cite:`Hill2015`.
Or see e.g. https://github.com/Adehill/KiD-A/tree/master/docs for its
documentation.

This test case is run via the Shipway and Hill 2012 example from PyMPDATA-examples, it is
slightly modified from :cite:`ShipwayHill2012`. Please refer to
https://github.com/open-atmos/PyMPDATA/tree/main/examples/PyMPDATA_examples/Shipway_and_Hill_2012
for the original source code.

See `tests/test_case_1dkid/test_pympdata_bulk.py <https://github.com/yoctoyotta1024/microphysics_testcases/blob/main/tests/test_case_1dkid/test_pympdata_bulk.py>`_
for an example of this test case working.
